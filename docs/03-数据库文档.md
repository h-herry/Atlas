## 1. 数据库概览 / Database Overview

### 1.1 数据库实例列表 / Database Instance List

| 数据库 / Database | 字符集 / Charset | 引擎 / Engine | 所属模块 / Module | 说明 / Description |
|---|---|---|---|---|
| atlas_common | utf8mb4 | InnoDB | common | 公共基础设施：i18n、消息、审计、组织架构、异步任务 |
| atlas_user | utf8mb4 | InnoDB | user | 用户服务与 RBAC 权限管理 |
| atlas_supplier | utf8mb4 | InnoDB | supplier | 供应商全生命周期管理：准入、评估、风险、协同、绩效 |
| atlas_purchase | utf8mb4 | InnoDB | purchase | 采购管理：9 种采购方式、订单、询报价、竞价 |
| atlas_contract | utf8mb4 | InnoDB | contract | 合同管理：模板、签署流程、履约跟踪、预警 |
| atlas_receipt | utf8mb4 | InnoDB | receipt | 收货结算：收货、ASN、三单匹配、对账 |
| atlas_material | utf8mb4 | InnoDB | material | 物料主数据：分类、属性模板、BOM、MRP、ERP 映射 |
| atlas_quality | utf8mb4 | InnoDB | quality | 质量管理：IQC、PPAP、NCR、检验标准、批次追溯 |

全局统一字符集 `utf8mb4`，全局统一引擎 `InnoDB`。 / Global charset `utf8mb4`, global engine `InnoDB`.

### 1.2 数据库配置说明 / Database Configuration Notes

所有数据库通过 Spring Boot 多数据源配置管理，各业务模块通过 `application.yml` 中 `spring.datasource.url` 指定对应数据库。迁移脚本由 Flyway 管理，分布在各模块 `src/main/resources/db/migration/` 目录下。
