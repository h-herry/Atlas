# 03 - 数据库文档 / Database Documentation

> 文档版本 / Doc Version：v1.2.21 | 最后更新 / Last Updated：2026-06-18

## 概述 / Overview

Atlas 项目共 8 个业务数据库 + 通用公共表，通过 Flyway 迁移脚本管理（V1 ~ V93）。SQL 声明表共 43 张，实体声明 77 个。所有表使用 InnoDB 引擎 + utf8mb4 字符集，主键采用雪花算法生成 BIGINT。

---

Atlas consists of 8 business databases + shared common tables, managed via Flyway migration scripts (V1 ~ V93). There are 43 SQL-declared tables and 77 entity declarations. All tables use the InnoDB engine + utf8mb4 charset, with Snowflake algorithm-generated BIGINT primary keys.

### 总体统计 / Overall Statistics

| 维度 / Dimension | 数量 / Count |
|------|:---:|
| SQL 迁移脚本 / SQL Migrations | 22 |
| CREATE TABLE 语句 / CREATE TABLE Statements | 43 |
| 去重后表总数 / Unique Tables | 39 张（不含 Flowable 引擎表） / 39 tables (excl. Flowable engine) |
| 实体声明数 / Entity Declarations | 77 |
| 数据库数量 / Database Count | 8 个独立数据库 + 通用表 / 8 independent + common |

## 11. 命名与主键规范 / Naming & Primary Key Conventions

| 项目 / Item | 规范 / Convention | 一致性 / Compliance |
|------|------|:---:|
| 表名 / Table Names | 小写 + 下划线 / lower_snake_case | 100% |
| 字段名 / Column Names | lower_snake_case | 100% |
| 主键 / Primary Key | BIGINT + `IdType.ASSIGN_ID`（雪花算法 / Snowflake） | 100% |
| 索引命名 / Index Names | `uk_*`（唯一 / unique）/ `idx_*`（普通 / normal） | 100% |
| 审计字段 / Audit Fields | `created_at` / `updated_at` DATETIME | 100% |
| 引擎 / Engine | InnoDB + utf8mb4 | 100% |
